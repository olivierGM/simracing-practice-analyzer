# Fichiers à NE PAS supprimer

## Structure du projet

### Version Production (Vanilla JS)
- `deploy/` - Site de production déployé
- `style.css` - Styles globaux de production
- `script-public.js` - Logique principale de production
- `src/` - Composants de production (pilot-card, driver-table, etc.)
  - Utilisés par deploy/index.html

### Version React (Migration)
- `frontend/` - Nouvelle version React/Vite
  - En développement, pas encore déployée
  - Utilise les mêmes données Firebase

### Ne pas supprimer
- `src/` (contient les composants de production)
- `style.css` (styles de production)
- `script-public.js` (logique de production)
