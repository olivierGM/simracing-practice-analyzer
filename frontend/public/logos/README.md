# Logos des marques automobiles

Ce dossier contient les logos SVG des marques automobiles utilisés dans l'application.

## Logos requis

Les logos suivants doivent être placés dans ce dossier :

- `audi.svg` - Logo Audi
- `ferrari.svg` - Logo Ferrari
- `mclaren.svg` - Logo McLaren
- `porsche.svg` - Logo Porsche
- `lamborghini.svg` - Logo Lamborghini
- `nissan.svg` - Logo Nissan
- `aston-martin.svg` - Logo Aston Martin
- `honda.svg` - Logo Honda
- `lexus.svg` - Logo Lexus
- `jaguar.svg` - Logo Jaguar
- `bentley.svg` - Logo Bentley

**Note:** Les logos BMW et Mercedes utilisent les URLs Wikimedia qui fonctionnent correctement.

## Sources recommandées

Vous pouvez télécharger ces logos depuis :

1. **Seeklogo** (https://seeklogo.com/) - Collection complète de logos en SVG
2. **Worldvectorlogo** (https://worldvectorlogo.com/) - Logos vectoriels gratuits
3. **Wikimedia Commons** (https://commons.wikimedia.org/) - Si disponibles

## Instructions

1. Téléchargez chaque logo au format SVG
2. Renommez-les selon la liste ci-dessus (en minuscules, avec tirets)
3. Placez-les dans ce dossier `/frontend/public/logos/`
4. Les logos seront automatiquement utilisés par l'application

## Dimensions recommandées

- Format : SVG (vectoriel)
- Couleur : Logos officiels des marques
- Taille d'affichage : 24x24px (automatique via CSS)
